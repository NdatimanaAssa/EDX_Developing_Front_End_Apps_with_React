# Developing Front End Apps with React - Cloud IDE Tools

This repository contains the code and resources for the **Developing Front End Apps with React** course, completed using the IBM Cloud IDE environment.

## Overview

This course teaches you how to build modern front-end applications with React. Using the IBM Cloud IDE, you learn to write, test, and deploy React apps in a browser-based development environment.

## What I Learned

- How to set up and work within the IBM Cloud IDE, including using its shell and file management commands.
- Building React components and organizing them within a structured project directory.
- Managing application state using React hooks and state management libraries like Zustand or Jotai.
- Implementing navigation in React apps with React Navigation.
- Using React Query or SWR for data fetching and caching.
- Understanding the limitations and capabilities of the Cloud IDE, such as no Git support and limited Python environment.
- Running and debugging React applications directly in the Cloud IDE.
- Best practices for structuring React projects for scalability and maintainability.

## Cloud IDE Tools

The IBM Cloud IDE provides:

- A zsh-like shell environment with essential commands (`ls`, `cd`, `mkdir`, `rm`, etc.)
- Support for Node.js and JavaScript development.
- Python 3 with standard libraries (no third-party packages).
- No native binary compilation or Git support.
- Ability to open and edit files via the `code` command.
- Support for React frameworks and libraries relevant to the course.

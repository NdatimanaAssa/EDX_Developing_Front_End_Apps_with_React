# Todo List

## Overview
A React application for managing a structured todo list with headings and nested list items under each heading.

## What I Learned

### useState Hook
- Managing multiple state variables: `todos`, `headingInput`, `listInputs`
- Using arrays and objects as state values
- Updating nested state with spread operator

### Dynamic Rendering
- Rendering lists dynamically using `Array.map()`
- Using `index` as key for mapped elements
- Conditionally rendering input values with controlled components

### Event Handling
- Handling input change events with `onChange`
- Triggering actions on button clicks (`Add Heading`, `Add List`, `Delete Heading`)
- Tracking per-item input state using an object keyed by index

## Features
- Add a heading to the todo list
- Add nested list items under each heading
- Delete any heading along with its list items
- Controlled inputs that clear after submission

## Technologies
- React 18
- Vite
- CSS

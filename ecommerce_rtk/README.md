# E-Commerce App with Redux Toolkit

## Overview
This project is an e-commerce application built with React and Redux Toolkit (RTK) as part of the **Developing Front End Apps with React** course on EDX.

## What I Learned

### Redux Toolkit (RTK)
- Setting up a Redux store using `configureStore`
- Creating slices with `createSlice` to manage state and reducers in one place
- Using `useSelector` to read state from the store
- Using `useDispatch` to dispatch actions to the store

### Cart Features
- Adding products to the cart
- Removing items from the cart
- Increasing and decreasing item quantity
- Clearing the entire cart
- Calculating the total amount

## Tools & Technologies
- React 18
- Redux Toolkit (RTK)
- Vite
- JavaScript (ES6+)
- CSS

# Content Rating

## Overview
A React component that allows users to like or dislike content and tracks the total number of ratings.

## What I Learned

### Class Components
- Building components using ES6 classes extending `Component`
- Using `constructor` and `super()` to initialize state

### State Management
- Managing `likes`, `dislikes`, and `totalRatings` in component state
- Updating state using `setState` with a previous state callback

### Event Handling
- Handling button click events to trigger state updates
- Storing handler functions directly inside state

## Features
- Like button that increments the like count
- Dislike button that decrements the dislike count starting from the current like value
- Total ratings counter that increments on every vote

## Technologies
- React 18
- Vite
- CSS

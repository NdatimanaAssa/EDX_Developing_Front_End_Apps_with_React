# Developing Front End Apps with React

## Course Overview
This repository contains all the projects and labs completed during the **Developing Front End Apps with React** course on EDX. The course covers building modern front-end applications using React and related tools.

## What I Learned

### React Fundamentals
- Understanding components and JSX syntax
- How React renders UI elements to the DOM

### Component State and Props
- Managing dynamic data within components using state
- Passing data between components through props

### Event Handling
- Responding to user actions like clicks and form inputs
- Creating interactive and dynamic applications

### Component Lifecycle
- How React components mount, update, and unmount
- Managing side effects during component lifecycle

## Projects

| Project | Description |
|---|---|
| `content_rating` | A like/dislike rating component with total ratings counter |
| `event_planner` | An event planning application with a footer component |
| `todo_list` | A todo list application for managing tasks |
| `feedback_form` | A feedback form with form validation and submission confirmation |
| `ecommerce_rtk` | An e-commerce app using Redux Toolkit for state management |
